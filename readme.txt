== Description ==
This plugin used as payment gateway for Ccavenue
This plugins required certain information about the Ccavenue like Merchant Id and Merchant Key which is given to when you register with Ccavenue for your payments
Please fill the information using admin option available in woocommerce payment setting.

== Installation ==
The quickest method for installing this plugin is:

upload the plugin by add new plugin option in wordpress
or just copy the plugin to the plugins directory inside wp-content in your wordpress directory.
